# mrdoc-desktop

基于 Electron 开发的 MrDoc 桌面客户端。

## 项目初始化

```
yarn install
```

或

```
npm install
```

## 运行项目

```
npm run electron.serve
```

或

```
yarn run electron.serve
```

## 打包项目

```
npm run electron:build
```

或

```
yarn run electron:build
```

### Lints and fixes files

yarn lint
